# AI Image & Video Generation System

An AI-powered application that generates high-quality images and videos from text prompts using Stable Diffusion and Hugging Face Diffusers. This project demonstrates the practical implementation of Generative AI models for creative content generation.

## Overview

This project enables users to generate realistic images and videos by simply providing text descriptions. It leverages state-of-the-art diffusion models to produce visually appealing and high-quality outputs.

## Features

* Text-to-Image Generation
* Text-to-Video Generation
* High-Quality Visual Outputs
* Custom Prompt Support
* Memory Optimization
* Google Colab Compatible
* Easy-to-Use Workflow

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/AI-Image-Video-Generation.git
cd AI-Image-Video-Generation
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the application:

```bash
python app.py
```

Enter a text prompt when requested and the model will generate an image based on the provided description.
